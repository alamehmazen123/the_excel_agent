v1.14.1 - Fixes the 'stuck on Auto-Generate' hang on large books (80k+ rows). Pivots are now built with calculation deferred (ManualUpdate) and Excel set to manual calc, so it no longer re-aggregates the whole sheet after every field; AutoFit no longer runs on the giant data sheet; the date is grouped before being moved into the Columns panel so it can't explode into day-columns; and wide Top-N hiding is capped. Measured: an 80k-row x 2-year book now finishes the full run in ~2 minutes (was hanging indefinitely).
